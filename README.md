# Exoplanet Data Storytelling

This project explores the Open Exoplanet Catalogue to uncover patterns, biases, and physical laws governing planetary systems. Through interactive and static visualizations, we investigate:
- Mass-radius relationship of exoplanets 
- Detection methods and their biases 
- Kepler's third law applied to exoplanetary systems 
- Habitable zone distribution 
- Hot Jupiters and their orbital characteristics 
- Stellar mass distribution in exoplanet surveys
